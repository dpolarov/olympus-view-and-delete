Olympus View — WiFi File Manager for Olympus & OM System Cameras

Manage, download and delete photos on Olympus and OM System cameras via WiFi — from your phone or computer. The free OI.Share alternative that actually works.

Android Windows Web Open Source

Why not OI.Share? Olympus View vs OI.Share

The official Olympus OI.Share app has serious limitations. Olympus View — the open-source OI.Share alternative — solves them:

  • Delete files from camera — OI.Share cannot delete photos
  • Batch download by date — OI.Share only allows one by one
  • RAW/ORF filter — OI.Share shows all duplicates
  • Works on Windows and in browser — OI.Share is mobile only
  • Open source — OI.Share is closed and no longer updated

Frequently Asked Questions

Can OI.Share delete photos from an Olympus camera?

No. The official OI.Share app cannot delete photos from your Olympus or OM System camera. Olympus View fills this gap — it supports direct file deletion from the camera memory card via WiFi.

How to delete photos from an Olympus camera via WiFi?

Enable WiFi on your camera → connect with Olympus View via QR code (Android) or direct WiFi (Windows/Web) → long-press a photo to enter selection mode → tap the delete button. Files are removed directly from the camera.

Does it work without installing anything?

Yes. The Web version works in any browser — no installation needed. Just connect your PC to the camera's WiFi hotspot and open the web app.

Technical Details

Stack

Flutter 3 / Dart — Material 3 — OPC Communication Protocol

Camera Protocol

The app communicates with the camera via HTTP over WiFi (IP: 192.168.0.10). Uses the OPC (Olympus OPC Communication Protocol).

QR Decoding

Olympus and OM System cameras use a proprietary QR code format (OIS1, OIS3) with a substitution cipher. The app decodes them automatically and extracts WiFi SSID, password and Bluetooth info.

Часто задаваемые вопросы

Можно ли удалить фото с камеры Olympus через OI.Share?

Нет. Официальное приложение OI.Share не умеет удалять фотографии с камеры. Olympus View восполняет этот пробел — поддерживает прямое удаление файлов с карты памяти камеры через WiFi.

Как удалить фото с камеры Olympus через WiFi?

Включите WiFi на камере → подключитесь через Olympus View по QR-коду (Android) или прямому WiFi (Windows/браузер) → зажмите фото для режима выбора → нажмите кнопку удаления. Файлы удаляются прямо с камеры.

Работает ли приложение без установки?

Да. Web-версия работает в любом браузере — установка не нужна. Подключите ПК к WiFi камеры и откройте веб-приложение.
