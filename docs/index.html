Olympus View

Manage photos on Olympus and OM System cameras via WiFi. Download, delete and browse files — from your phone or computer.

Android Windows Web Open Source

Why not OI.Share?

The official Olympus OI.Share app has serious limitations. Olympus View solves them:

  • Delete files from camera — OI.Share cannot delete photos
  • Batch download by date — OI.Share only allows one by one
  • RAW/ORF filter — OI.Share shows all duplicates
  • Works on Windows and in browser — OI.Share is mobile only
  • Open source — OI.Share is closed and no longer updated

RAW Filter

Hide ORF/RAW files with one button to remove duplicates when shooting RAW+JPG.

Download groups of files with progress. On Android photos appear in gallery immediately.

How to Use

  1. Enable WiFi on the Olympus / OM System camera
  2. On Android: open the app and scan the QR code from the camera screen
  3. On Windows: connect to the camera WiFi, then open the app
  4. The photo list loads automatically with thumbnails
  5. Long press on a photo — enter file selection mode
  6. Use buttons for batch selection, download and deletion

Technical Details

Stack

Flutter 3 / Dart — Material 3 — OPC Communication Protocol

Camera Protocol

The app communicates with the camera via HTTP over WiFi (IP: 192.168.0.10). Uses the OPC (Olympus OPC Communication Protocol).

QR Decoding

Olympus and OM System cameras use a proprietary QR code format (OIS1, OIS3) with a substitution cipher. The app decodes them automatically and extracts WiFi SSID, password and Bluetooth info.

Чому не OI.Share?

Офіційний додаток Olympus OI.Share має серйозні обмеження. Olympus View вирішує їх:

  • Видалення файлів з камери — OI.Share не вміє видаляти фото
  • Масове завантаження за датами — OI.Share дозволяє лише по одному
  • Фільтр RAW/ORF файлів — OI.Share показує всі дублі
  • Працює на Windows та у браузері — OI.Share лише мобільний
  • Відкритий вихідний код — OI.Share закритий і більше не оновлюється

Почему не OI.Share?

Официальное приложение Olympus OI.Share имеет серьёзные ограничения. Olympus View решает их:

  • Удаление файлов с камеры — OI.Share не умеет удалять фото
  • Массовое скачивание по датам — OI.Share позволяет только по одному
  • Фильтр RAW/ORF файлов — OI.Share показывает все дубли
  • Работает на Windows и в браузере — OI.Share только мобильное
  • Открытый исходный код — OI.Share закрытый и больше не обновляется
